# QuickOCR ユーザーガイド

## はじめに

QuickOCRは、Macの画面上のあらゆるテキストを瞬時に読み取り、クリップボードにコピーするアプリケーションです。画像、PDF、動画内のテキストなど、選択できないテキストも簡単にコピーできます。

## インストールと起動

1. **アプリの準備**:
   配布された `.dmg` ファイルを開き、`QuickOCR.app` を「アプリケーション」フォルダにドラッグ＆ドロップしてください。また、ソースコードからビルドした場合は、ビルド出力フォルダからアプリを移動します。

2. **初回起動**:
   アプリをダブルクリックして起動します。
   初回起動時に「画面収録」の権限を求められます。これは画面上のテキストを読み取るために必須の権限です。「システム設定を開く」をクリックし、QuickOCRのチェックボックスをオンにしてください。

3. **常駐**:
   起動するとメニューバーにアイコン（[T] のようなアイコン）が表示されます。アプリはバックグラウンドで常駐し、いつでも呼び出せます。

## 基本的な使い方

### 1. テキストを読み取る (OCR)

1. **ショートカットキー** `Cmd + Shift + O`（デフォルト）を押します。
2. 画面が少し暗くなり、マウスカーソルが十字に変わります。
3. 読み取りたいテキストを**マウスでドラッグ**して囲みます。
4. マウスを離すと、自動的にテキスト認識が始まり、クリップボードにコピーされます。

### 2. 結果の確認

- 読み取りが完了すると通知が表示されます。
- `Cmd + V` でメモ帳やエディタにテキストを貼り付けることができます。

### 3. キャンセル

- 範囲選択中に `Esc` キーを押すか、マウスを右クリックするとキャンセルできます。

## 設定

メニューバーのアイコンをクリックし、「設定...」を選択すると設定画面が開きます。

- **ログイン時に自動起動**: Macを起動したときに自動的にQuickOCRを開始します。
- **音声フィードバック**: 読み取り完了時にシステムサウンド（Pingなど）を再生します。
- **テキスト整形**: 読み取ったテキストの余分な改行や空白を自動的に削除・整理します。
- **ショートカットキー**: 起動用のショートカットキーを変更できます。入力ボックスをクリックして、新しいキーの組み合わせを押してください。

## アンインストール

アプリケーションフォルダから `QuickOCR.app` をゴミ箱に入れてください。
設定ファイルは `~/Library/Preferences/` 内に保存されていますが、そのままでも問題ありません。
