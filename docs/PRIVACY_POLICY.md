# プライバシーポリシー

**最終更新日: 2026年2月4日**

QuickOCR（以下、「本アプリ」）における利用者情報の取り扱いについて、以下の通り定めます。

## 1. 情報の収集と使用

**本アプリは、利用者の個人情報を収集・送信しません。**

- **OCR処理**: 画面上のテキスト認識（OCR）は、すべてお客様のMacデバイス内（ローカル）で行われます。画像データや認識されたテキストが外部サーバーに送信されることは一切ありません。
- **設定データ**: アプリの設定（ショートカットキーなど）はデバイス内に保存されます。

## 2. 必要な権限

本アプリは機能提供のために以下の権限を使用しますが、これらは記述された目的以外には使用されません。

- **画面収録 (Screen Recording)**: ユーザーが選択した画面領域の画像をキャプチャし、テキスト認識を行うために使用します。常にユーザーの操作（矩形選択）に基づいてのみ実行されます。

## 3. 免責事項

本アプリの使用によって生じた、いかなる損害についても開発者は責任を負いません。

## 4. お問い合わせ

本ポリシーに関するご質問は、GitHubリポジトリのIssuesまでお願いいたします。
