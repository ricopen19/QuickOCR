# リリースノート

## v1.0.0 (2026-02-04)

QuickOCRの最初の公式リリースです。Mac画面上のあらゆるテキストを瞬時にキャプチャしてコピーできます。

### 機能一覧

#### 🚀 コア機能

- **矩形選択OCR**: `Cmd+Shift+O` で即座に範囲選択モードを起動。
- **高速テキスト認識**: Vision Frameworkを使用した高速かつ高精度なOCR（日本語・英語対応）。
- **クリップボード連携**: 認識結果を自動的にクリップボードにコピー。
- **通知機能**: 完了時に認識結果のプレビューを通知。

#### ⚙️ 設定・カスタマイズ

- **ショートカットキー変更**: ユーザー好みのキーバインドに設定可能。競合チェック機能付き。
- **自動起動**: ログイン時に自動的にアプリを起動するオプション。
- **音声フィードバック**: 完了時にサウンドを再生するオプション。
- **テキスト整形**: 読み取ったテキストの不要な改行を除去するスマートフォーマット機能。

#### 🛡️ その他

- **オンボーディング**: 初回起動時にわかりやすいチュートリアルと権限設定ガイドを表示。
- **プライバシー配慮**: すべての処理をデバイス内で完結（オフライン動作）。外部通信なし。

---

**システム要件**: macOS 14.0 (Sonoma) 以降
